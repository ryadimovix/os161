Cara Membuat Akun GitHub


Pada kesempatan kali ini saya akan menjelaskan cara membuka akun GitHub dan cara  menggunakan GitHub.  GitHub adalah tempat upload project - project opensource dimana project tersebut bisa dikembangkan oleh programmer lain, hal ini baik sekali bila kita memilki project dengan sebuah team bisa dikatakan juga sosial medianya programmer. Kelebihan github adalah ketika kita membuat sebuah repository project kemudian menguploadnya lalu kita membuat perubahan atau penambahan pada project itu dan di upload ulang ke repository itu maka project yang lama masih di simpan dan tidak akan hilang, jadi kita bisa membuat versi - versi dari project yang kita buat dan hal ini sangat memudahkan kita bila kita ingin ke versi sebelumnya.

Berikut ini adalah tampilan ketika akan membuka akun GitHub, yang dibutuhkan adalah alamat email Anda untuk mendaftar.

1. Buka www.github.com pada browser Anda, Klik Sign up

2. Kemudian lakukan langkah-langkah yang telah diterangkan pada gambar di bawah ini 


3. Setelah meng-klik tombol Create Account, Anda masuk ke jendela pilihan personal plan, silahkan pilih "unlimited public repositories for free" untuk akun yang tidak berbayar. Dan apabila Anda ingin yang akun yang private, silahkan pilih yang berbayar.  Kemudian klik Finish sign up.


4. Akun GitHub telah selesai dibuat, namun Anda harus verifikasi akun Anda yang akan dikirimkan pihak Github melalui email yang telah Anda daftarkan. Setelah konfirmasi, Anda sudah dapat membuat Repository.
